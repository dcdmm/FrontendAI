CSS 定位（Positioning）完全指南

1. static（默认值 —— top/left 无效）

所有元素默认 position: static，按文档流从上往下排列，设置 top/left 等属性无效。

2. relative（相对自身原位置偏移，原位置仍占空间）

我偏移了

绿色方块设置了 position: relative; top: 30px; left: 50px;
它相对于自己原来的位置移动了，但原位置的空间仍然保留（蓝色没有顶上去）。

3. absolute（相对于最近的定位祖先，脱离文档流）

HOT 父元素设置了 position: relative

absolute 元素会找最近的 position 不是 static 的祖先作为参考。
所以常见搭配：父 relative + 子 absolute。

4. fixed（相对于浏览器窗口，滚动时不动）

右下角的红色圆形按钮就是 position: fixed，无论怎么滚动页面它都固定在那里。

常见用途：固定导航栏、回到顶部按钮、悬浮客服图标。

sticky = relative + fixed 的结合：没到阈值时是 relative，滚到阈值后变成 fixed。

6. z-index（控制层叠顺序，值越大越在上面）

z-index 只对 position 不是 static 的元素有效。
