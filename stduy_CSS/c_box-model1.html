1. 什么是 Flexbox？

Flexbox（弹性盒子）是 CSS3 中一种一维布局模型，专门用来在一个方向上（水平或垂直）分配空间和对齐元素。

核心概念：给父容器设置 display: flex，它就变成了弹性容器（flex container），其直接子元素自动变成弹性项目（flex items）。

Flexbox 有两条轴：

  • 主轴（Main Axis）：默认是水平方向（从左到右）
  • 交叉轴（Cross Axis）：与主轴垂直的方向（默认从上到下）

2. flex-direction — 主轴方向

6. 子项属性 — flex-grow / flex-shrink / flex-basis

这三个属性设置在子元素上，控制它们如何分配剩余空间。

flex-grow — 放大比例

定义项目的放大比例，默认为 0（不放大）。数值越大，分到的剩余空间越多。

第2项 flex-grow:1（黄色），第3项 flex-grow:2（绿色）→ 绿色分到的空间是黄色的2倍
固定
grow:1
grow:2

flex-shrink — 缩小比例

定义项目的缩小比例，默认为 1（等比缩小）。设为 0 表示不缩小。

容器宽300px，每项宽200px → 第2项 flex-shrink:0（黄色）不缩小，其他项被压缩

flex-basis — 初始大小

设置项目在分配多余空间之前的初始大小，替代 width。

三个项目的 flex-basis 分别为 100px、200px、50px

flex 简写

常用简写：flex: 1 等于 flex: 1 1 0%（均分空间）
flex:1、flex:2、flex:1 → 中间项占 2/4 = 50% 的空间

8. order — 排列顺序

默认所有项目的 order 为 0，数值越小越靠前。可以不改变 HTML 结构来重新排序。

HTML 顺序是 1、2、3，但 order 设为 3、1、2 → 显示顺序变为 2、3、1
1 (order:3)
2 (order:1)
3 (order:2)

10. 属性速查表

容器属性（设在父元素上）

属性作用常用值
display开启弹性布局flex / inline-flex
flex-direction主轴方向row / column
justify-content主轴对齐center / space-between / space-evenly
align-items交叉轴对齐center / stretch / flex-start
flex-wrap是否换行nowrap / wrap
gap项目间距10px / 1rem
align-content多行时的交叉轴对齐同 justify-content

子项属性（设在子元素上）

属性作用常用值
flex-grow放大比例0（默认）/ 1
flex-shrink缩小比例1（默认）/ 0
flex-basis初始大小auto / 200px
flex以上三者的简写flex: 1 / flex: 0 0 auto
align-self单独交叉轴对齐center / flex-end
order排列顺序0（默认）/ 任意整数
