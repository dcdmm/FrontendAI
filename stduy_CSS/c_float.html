例4: 文字环绕 — float 最经典的用途

假装是图片

这是一段很长的文字, 用来演示浮动的经典效果——文字环绕。 当一个元素设置了 float: left 后, 它会脱离正常的文档流, 但周围的文字内容会自动环绕在它的周围, 就像报纸排版一样。 这其实就是 float 最初被设计出来的目的。 我们再多写一些文字来让效果更明显。 浮动让元素像水中的浮标一样"浮"起来, 而文字就像水流一样绕过它继续流动。 这个效果非常直观, 也是理解浮动的最好方式。
